CSC101: Internet Technology & Programming I

An outline of computer networks, internet technologies, and the fundamentals of web markup languages.

Module 1: Computer Network Fundamentals

  • Basic Concepts: Definition of a computer network, nodes, and links.
  • Key Terminology:
    • Protocol: Rules for data transmission (e.g., TCP/IP, HTTP).
    • Topology: Physical and logical arrangement of a network (e.g., Bus, Star, Ring, Mesh).
    • IP Address: Unique numerical identifier for a device on a network.
    • DNS: Domain Name System, translates domain names to IP addresses.
    • Firewall: A security device for monitoring and controlling network traffic.
  • Types of Computer Networks:
    • LAN (Local Area Network): Covers a small area like an office or home.
    • WAN (Wide Area Network): Covers a large geographical area, connecting LANs.
    • Cloud Networks: On-demand networks hosted by service providers.
  • Network Architecture:
    • Client-Server: Central server manages client nodes.
    • Peer-to-Peer (P2P): No central server; every node can act as a client or server.
  • The OSI Model: The 7 layers of network communication (Physical, Data Link, Network, Transport, Session, Presentation, Application).

Module 2: The Internet and World Wide Web

  • Definition of the Internet: A global, decentralized "network of networks".
  • History of the Internet: From ARPANET (1960s) to TCP/IP and the modern web.
  • Uses of the Internet: E-commerce, Cashless Transactions, Education, Social Networking, Entertainment.
  • Security & Privacy: Understanding risks (hacking, data breaches) and protection steps (antivirus, strong passwords, VPNs).
  • World Wide Web (WWW): A service that runs on the internet to provide access to web pages and applications.
  • Key Differences: Internet is the hardware/infrastructure; WWW is the software/information service that uses it.
  • WWW Architecture: Components include Clients (browsers), Servers, URL, HTTP/HTTPS, and HTML.

Module 3: Markup Languages & HTML

  • Definition of a Markup Language: A system for annotating a document to define its structure and presentation.
  • HTML (HyperText Markup Language): The standard language for creating web pages.
  • Basic HTML Structure & Tags: html, head, body, title, h1-h6, p, etc.
  • Key Elements:
    • Anchor Tag (a): Creates hyperlinks.
    • Lists (ul, ol, li): Creates bulleted or numbered lists.
    • Tables (table, tr, th, td): Organizes data in rows and columns.
    • Image Tag (img): Embeds images.
    • Forms (form, input): Collects user input.
  • Physical vs. Logical Tags:
    • Physical Tags (e.g., b, i): Define visual appearance. (Considered outdated).
    • Logical Tags (e.g., strong, em): Define semantic meaning or structure. (Modern best practice).
  • Other Markup Languages: A brief overview of XML, XHTML, and SGML.
